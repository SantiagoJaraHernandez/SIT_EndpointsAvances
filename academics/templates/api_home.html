¿Qué es el Sistema de Intervención Temprana?

SIT es un sistema diseñado para ayudar a profesionales y centros educativos a gestionar de forma eficaz los programas de intervención temprana. Con una API robusta y segura, permite controlar información de cursos, asignación de docentes, matrícula de estudiantes y seguimiento de tareas, todo accesible vía servicios REST.

Funcionalidades principales

Gestión de Cursos

Crear, editar y listar cursos de intervención con fechas y responsables asignados.

Manejo de Estudiantes

Registro seguro y seguimiento individual de estudiantes activos en programas.

Autenticación Segura

Sistema de login y logout basado en tokens JWT para máxima seguridad.

Gestión de Tareas

Creación, edición y seguimiento de tareas asignadas a estudiantes y docentes.

¿Cómo empezar a usar la API?

  1. Obtén tu token de autenticación registrándote o con tu cuenta asignada.
  2. Usa el endpoint base /api/ para acceder a los recursos.
  3. Consulta la documentación y prueba los endpoints en nuestra interfaz Swagger.
  4. Incluye el token JWT en el encabezado Authorization: Bearer <token> para las peticiones protegidas.
